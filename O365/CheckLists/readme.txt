MOSS_ExportLists.ps1

Used for export all lists information (WebURL,Title,Url,LastModified,ItemCount,Type,ItemsFilePath) from entire site collection.

Requirements:

This script must be executed on MOSS application server. 

Parameters:

-MossSiteUrl - URL of MOSS site collection, example - http://mysite

Output:

Csv file (MOSS_<SiteCollectionName>_Lists.csv) saved on the script execution location - it contains all lists metadata.
This file is used as input for second script - SPO_ExportLists.ps1

Csv files saved in subfolder named MOSS_Libraries in script location - each of them contains document library items (documents) metadata such as Name,RelativeUrl,Url,Modified.
Files name format is next:

<SiteCollectionName>_<DocumentLibraryTitle>.csv


Usage:

.\MOSS_ExportLists.ps1 -MossSiteUrl http://mysite

SPO_ExportLists.ps1

Used to check MOSS lists migrated to O365 for missing items and export documents libraries items metadata.

Requirements:

Script requires PowerShell v.3.0 installed

Parameters:

-MossListsFile - Full path to file, generated by MOSS_ExportLists.ps1 script. Example - c:\Scripts\MOSS_MySite_Lists.csv

-O365SiteUrl - URL of Office365 SharePoint site collection where MOSS lists were migrated. Example - https://mysite.sharepoint.com

-UserName - Office365 user name with Site Collection permissions on related O365 site collection Example - myusername@mytenant.onmicrosoft.com

-UserPassword -Office365 user password

Output:

2 Csv files saved on the script execution location:
 
 SPO_<SiteCollectionName>_ListsWithDiffItems - contains all MOSS lists founded on O365 with different ItemCount (O365 list ItemCount not equal MOSS list ItemCount)
 SPO_<SiteCollectionName>_MissedLists - contains all MOSS lists not founded on O365 (missed)
 
Csv files saved in subfolder named SPO_Libraries in script location - each of them contains document library items (documents) metadata such as Name,RelativeUrl,Url,Modified.
Files name format is next:

<SiteCollectionName>_<DocumentLibraryTitle>.csv 
 
Usage:

.\SPO_ExportLists.ps1 -MossListsFile c:\Scripts\MySite_Lists.csv -O365SiteUrl https://mysite.sharepoint.com -UserName myusername@mytenant.onmicrosoft.com -UserPassword *****

CompareLibDocuments.ps1

Used to compare MOSS document libraries items versus SPO documents for each library.

Requirements:

Script requires PowerShell v.3.0 installed

Parameters:

-MossExportFolder  - Full path to folder contains MOSS libraries csv files (generated using MOSS_ExportLists script)
-SPOExportFolder - Full path to folder contains SPO libraries csv files (generated using SPO_ExportLists script)

Output:

Csv files saved in subfolder named Reports in script location - there can be 2 files for each document library:
<SiteCollectionName>_<DocumentLibraryTitle>_missed.csv - contains missed documents (Exist on MOSS but not found on SPO)
<SiteCollectionName>_<DocumentLibraryTitle>_modified.csv - contains documents with different Modified date (MOSS Modified date not equal SPO Modified date)

Usage:
.\CompareLibDocuments.ps1 -MossExportFolder c:\Scripts\MOSS_Libraries -SPOExportFolder c:\Scripts\SPO_Libraries
